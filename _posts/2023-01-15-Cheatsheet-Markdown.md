# Markdown Cheatsheet

## Titles
``` markdown
# H1
## H2
### H3
#### H4
##### H5
###### H6

H1
===
H2
---
```
> # H1 Title
> H1 Title
===
> ## H2
> H2 Title
---
> ### H3
> #### H4
> ##### H5
> ###### H6

## Text
``` markdown
**bold text**
__blod text__
*tilicized text*
_tilicized text_
```
> **bold text**  
> __blod text__  
> *tilicized text*  
> _tilicized text_

## Lists :
### Ordered
``` markdown
1. First item
2. Second item
```
### Unordered
``` markdown
- First item
- Second item
```

## Blockquote

``` markdown
> Blockquote
```

## Code

``` markdown
`dev`
\``` # Remove the back slash before use
bloc of code
\``` # Remove the back slash before use
```

## Horizontal rule
``` markdown
---
```

## Link & Images
``` markdown
[text](https://www.example.com) # Link
![alt text](image.jpg) # Image
```
